# Calculate.All
Calculate.All is a simple and user-friendly web application that provides multiple online calculation tools in one place. This project is built using HTML, CSS, and JavaScript, where the user interface is designed with HTML and CSS, and all calculation logic is implemented using JavaScript.

# Features
.Age Calculator
  Calculates the user’s age based on their date of birth.

.BMI Calculator
  Calculates Body Mass Index (BMI) using height and weight and shows the result clearly.

.Final Grade Calculator
  Helps students calculate their final grade based on given marks and grading logic.

.Weighted Grade Calculator (Coming Soon)
  A weighted grade calculator is currently under development and will be added soon.

# Technologies Used
.HTML – Structure and layout

.CSS – Styling and responsive design

.JavaScript – Core logic, calculations, and dynamic interactions
